# Cricket Analysis

Cricket Analysis is a simple application that allows users to search for cricket players and view their statistics. It uses an XML file as a data source for player information.

## Features

- Search for cricket players by name
- Display player statistics including matches, innings, runs, average, and strike rate

## Technologies Used

- Python
- Streamlit
- XML

## Installation

1. Clone the repository:
git clone https://github.com/AkshayPatil20001/Cricket-stat-analysis-individual-player.git


2. Install the required dependencies:

pip install -r requirements.txt


3. Run the application:

streamlit run app.py

The application will start running on a local server, and you can access it in your web browser at http://localhost:8501.

## Usage

1. Enter the name of the cricket player you want to search for in the search input field.
2. Click the "Search" button.
3. If the player is found, their statistics will be displayed on the screen.
4. You can search for different players by repeating the above steps.
